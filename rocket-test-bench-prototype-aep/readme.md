# Rocket Test Bench Prototype (AEP Internship)

## Overview
Development of a prototype test bench for rocket propulsion experiments during an internship at the Paraguay Space Agency (AEP).

## Contributions
- Mechanical design of structural components
- Support in control system integration
- Thermal analysis for a custom heat exchanger (liquid oxygen generation)
- Coordination with suppliers for materials and tools

## Technologies
- Mechanical design
- Thermal systems
- Experimental setup development

## Status
Prototype development phase (project not fully completed)
