# Solar-Powered Mosquito Trap

## Overview
Design and implementation of a solar-powered mosquito trap system for sustainable pest control.

## Contributions
- Design of energy management system
- Arduino-based control implementation
- Circuit design for solar power integration

## Technologies
- Arduino
- Embedded systems
- Solar energy systems
- Electronics design

## Results
- Awarded "Best Design Award" in university project evaluation
