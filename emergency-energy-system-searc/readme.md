# SEARC – Emergency Energy System

## Overview
Design and development of a rechargeable domestic energy system for power outages.

## Contributions
- Control system design using PLC
- Electrical power system design (motors, alternator, inverter)
- Mechanical fabrication and machining
- Integration of system components

## Technologies
- Siemens PLC
- Electrical circuits
- Power systems
- Machining

## Application
Backup power system for residential use
